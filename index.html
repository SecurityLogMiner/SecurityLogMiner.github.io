Log Collection System

Log Collection System provides the insight needed to secure, troubleshoot, and optimize systems and applications. Whether it is an individual user or larger organization, log collection is the first step in the analysis process. The collection and storage of system and application logs is designed with ease-of-use in mind to provide simple and efficient event visibiilty.

About Project

Log Collection System emerged as a capstone project at Oregon State University. Starting as a security project, the team consists of Joeseph Murche and Ivan Chan. The project is advised by project partner Casey Colley.
